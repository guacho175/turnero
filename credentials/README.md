# Credenciales Google Calendar

Este proyecto requiere autenticación OAuth con Google Calendar.

## Archivos requeridos en ejecución

- credentials.json
- token.json

## Cómo obtenerlos

1) Crear credenciales OAuth en Google Cloud Console
2) Descargar el archivo y guardarlo como:
   credentials/credentials.json
3) Ejecutar el proyecto y autorizar el acceso
4) El archivo token.json se generará automáticamente

Estos archivos NO se versionan.
